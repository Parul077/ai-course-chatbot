# AI Course Support Chatbot

An AI-powered chatbot that answers student queries about courses using OpenAI APIs.

## Features

- AI-powered question answering
- Session-based chat interaction
- Clean chat UI
- Flask API backend
- Prompt-controlled responses

## Tech Stack

Backend:
- Python
- Flask

AI:
- OpenAI API

Frontend:
- HTML
- CSS
- JavaScript

## Run Locally

Clone the repository

pip install -r requirements.txt

python run.py
